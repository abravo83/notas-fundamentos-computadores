# Tema 2: Representación de la información

Dentro de un computador, cualquier información está representada como una cadena de 0's y 1's. Sin embargo, una cadena de ceros y unos no tiene sentido si no sabemos cómo está codificada esa información: el tipo de dato y el sistema usado para representar a ese tipo de dato (el sistema de numeración si se trata de un número, la tabla de codificación de caracteres si se trata de texto, el algoritmo de codificación si es multimedia, etc.)
